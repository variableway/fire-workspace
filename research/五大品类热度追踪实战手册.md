# 五大品类热度追踪实战手册

> 本文提供香水、包袋、护肤品、手表、球鞋五个核心品类的**每日/每周标准操作流程（SOP）**。你不需要懂技术，只需要按步骤打开对应的APP/网站，记录关键数据，就能建立对热度变化的敏感度。
>
> 每个品类包含：**信号源清单 → 每日5分钟流程 → 每周30分钟深度扫描 → 关键指标阈值 → 实战案例**

---

## 品类一：香水（Fragrance）

### 为什么香水值得追踪

香水是**热度传导最快、季节性最强、男女交叉最明显**的品类。一款新香可以在2周内从小红书爆红到全网断货，也可以在一个季度后沦为"时代眼泪"。

### 信号源清单

| 优先级 | 平台 | 看什么 | 入口 |
|--------|------|--------|------|
| P0 | **小红书** | 新品试香笔记、"高级感香水"话题、"求推荐"帖的评论区高赞 | 搜索"香水推荐""XX香测评" |
| P0 | **Fragrantica** | 新香入库速度、用户评分变化、评论数增长 | fragrantica.com |
| P1 | **香水时代** | 国内香友讨论、小众香评测 | nosetime.com |
| P1 | **中免日上/丝芙兰** | 畅销榜、断货状态 | APP内榜单 |
| P2 | **闲鱼** | 余量瓶流通速度、分装价格 | 搜索"XX香 余量""XX香 分装" |
| P2 | **淘宝搜索建议** | 输入品牌名看下拉联想词 | 淘宝APP搜索框 |

### 每日5分钟流程

```
Day Check（每天任意时间）：

1. 打开小红书，搜索"香水推荐"，筛选"最热"
   └─ 记录：前10篇笔记中，有多少篇提到了同一个品牌/香型？
   └─ 记录：是否有你之前没听说过的新名字？

2. 打开中免日上APP，看"香水畅销榜"TOP 20
   └─ 记录：榜单前5名是否有变化？
   └─ 记录：是否有之前不在榜上的新面孔？

3. 打开淘宝，输入"香水"，看下拉联想词前5个
   └─ 记录：用户实际在搜什么？（这是真实需求）
```

### 每周30分钟深度扫描

```
Week Scan（建议周日晚上）：

1. 新香扫描（10分钟）
   ├─ 打开 Fragrantica，看"New Fragrances"本周入库
   ├─ 打开香水时代，看首页新品推荐
   └─ 记录：哪些新香同时在中外平台被提及？

2. 口碑验证（10分钟）
   ├─ 选一款本周发现的新香/热香
   ├─ 小红书搜索它，看50篇以上笔记
   ├─ 区分：素人分享 vs KOL推广（看粉丝数和内容深度）
   └─ 判断：留香时间、场合适配、是否"街香"的真实反馈

3. 价格锚定（5分钟）
   ├─ 查中免日上价格
   ├─ 查FragranceNet价格
   ├─ 查闲鱼余量瓶价格
   └─ 计算：国内外价差、二手折旧率

4. 趋势记录（5分钟）
   └─ 更新追踪表：本周最热香型/品牌、验证结论、下周关注
```

### 关键指标与阈值

| 指标 | 低热度 | 中热度 | 高热度 | 过热/泡沫 |
|------|--------|--------|--------|----------|
| 小红书周新增笔记数 | <50 | 50-200 | 200-1000 | >1000 |
| 素人内容占比 | <30% | 30-60% | 60-80% | >80%（可能自然发酵） |
| 闲鱼余量瓶数量 | 极少 | 少量 | 大量 | 极大量（热度消退信号） |
| 中免日上排名变化 | 无变化 | 上升5位内 | 冲入前10 | 断货 |
| 国内外价差 | <20% | 20-40% | 40-60% | >60%（套利空间或区域限定） |

### 男女交叉追踪点

- **男性香女性化**：追踪"女生用男香""男香女穿"话题下，Dior旷野、Chanel蔚蓝、Terre d'Hermès被女性提及的频率
- **中性香崛起**：追踪Le Labo、Byredo、Diptyque、Maison Margiela的性别标签变化
- **送礼流向**：追踪"送男朋友香水""送女朋友香水"话题的品类差异

### 实战案例

**案例：判断"马吉拉 Replica 系列"是真火还是营销**

| 验证步骤 | 操作 | 结果 | 判断 |
|----------|------|------|------|
| 搜 | 微信指数查"Maison Margiela""Replica" | 近90天持续上升 | 趋势真实 |
| 看 | 小红书搜"Replica"，前50篇笔记 | 素人占比约70%，内容有深度（场景描述、留香反馈） | 真实UGC |
| 价 | 闲鱼搜"Replica 余量" | 余量瓶价格接近原价8折，流通快 | 真实需求 |
| 问 | 问3位女性朋友是否知道 | 2位知道，1位有入手 | 认知度较高 |
| 等 | 追踪3个月 | 热度持续，新品（如"慵懒周末"）持续被讨论 | 长线热度 |

**结论**：真热度，非营销泡沫。适合作为送礼安全牌或自用入门。

## 品类二：包袋（Handbags）

### 为什么包袋值得追踪

包袋是**品牌层级最清晰、二手市场最成熟、保值属性最明确**的品类。追踪包袋热度，本质上是在追踪**社交货币的流通状态**。

### 信号源清单

| 优先级 | 平台 | 看什么 | 入口 |
|--------|------|--------|------|
| P0 | **小红书** | 上身图、容量测评、"第一只大牌包"话题 | 搜索"XX包测评""最值得买的包" |
| P0 | **红布林** | 热销榜、成交速度、价格趋势 | APP首页榜单 |
| P1 | **品牌官网/小程序** | 新品发布、断货状态、配货政策 | 各品牌官方渠道 |
| P1 | **只二/妃鱼** | 二手成交价、上架速度 | APP内搜索 |
| P2 | **PurseBlog（英文）** | 国际包友讨论、保值率分析 | purseblog.com |
| P2 | **闲鱼** | 个人闲置流转速度、求购帖数量 | 搜索"XX包 闲置" |

### 每日5分钟流程

```
Day Check：

1. 打开小红书，搜索"包包推荐"，筛选"最热"
   └─ 记录：前10篇中，哪几个品牌出现频率最高？
   └─ 记录：是否有"断货""溢价""买不到"等关键词？

2. 打开红布林，看"今日热销"和"新晋热门"
   └─ 记录：二手市场什么在快速流通？

3. 打开1-2个品牌官网/小程序（轮巡）
   └─ 记录：是否有上新？是否有"即将售罄"标记？
```

### 每周30分钟深度扫描

```
Week Scan：

1. 新品追踪（10分钟）
   ├─ 看LV、Chanel、Hermès、Dior、Gucci官网本周上新
   ├─ 小红书搜索"XX品牌 新品"
   └─ 记录：哪几款在社交媒体被快速讨论？

2. 二手市场验证（10分钟）
   ├─ 选一款本周热门包，查红布林/只二的成交数据
   ├─ 查闲鱼的"求购"帖数量（求购多=需求旺）
   ├─ 对比专柜价、二手价、欧洲退税价的价差空间
   └─ 判断：保值率、流通速度

3. 长期趋势观察（5分钟）
   ├─ 追踪经典款（Neverfull、CF、Birkin）的价格稳定性
   ├─ 对比季节款 vs 经典款的二手折价差异
   └─ 记录：什么包型正在上升（如近年托特包、Hobo包回潮）

4. 更新追踪表（5分钟）
```

### 关键指标与阈值

| 指标 | 低热度 | 中热度 | 高热度 | 过热/泡沫 |
|------|--------|--------|--------|----------|
| 小红书周新增笔记 | <30 | 30-100 | 100-500 | >500 |
| 红布林平均售出天数 | >60天 | 30-60天 | 15-30天 | <7天（秒出） |
| 二手溢价率 | 折价>30% | 折价10-30% | 原价±10% | 溢价>20% |
| 专柜状态 | 随时可买 | 部分颜色缺货 | 全面断货 | 需配货/排队 |
| 求购帖/出售帖比 | <0.2 | 0.2-0.5 | 0.5-1.0 | >1.0（供不应求） |

### 男女交叉追踪点

- **男包女性化**：追踪LV Soft Trunk、Gucci 1955、Prada Re-Nylon被女性使用的频率
- **女包男性化**：追踪Loewe Puzzle、Celine Triomphe被男性使用的趋势
- **中性包崛起**：追踪Longchamp、Goyard、Freitag等"无性别"品牌的讨论量

### 实战案例

**案例：判断"Miu Miu 近年是否翻红"**

| 验证步骤 | 操作 | 结果 |
|----------|------|------|
| 搜 | 百度指数"Miu Miu"近2年趋势 | 从2023年开始持续上升 |
| 看 | 小红书搜"Miu Miu包"，看内容类型 | 大量穿搭笔记，不仅限于包，还有鞋、服装 |
| 价 | 红布林搜Miu Miu经典款 | 二手价格坚挺，甚至接近原价 |
| 问 | 问时尚圈朋友 | "Miu Miu现在是'少女美学'代表，张元英效应" |
| 等 | 追踪6个月 | 热度持续，品牌从"Prada副线"变成独立热门 |

**结论**：真热度，品牌定位转型成功。但需注意：Miu Miu的季节款仍不保值，经典款（如Wander包）才是硬通货。

## 品类三：护肤品（Skincare）

### 为什么护肤品值得追踪

护肤品是**复购率最高、成分党最活跃、季节性最明显**的品类。热度不仅来自品牌，更来自**成分**（如烟酰胺、A醇、二裂酵母）。

### 信号源清单

| 优先级 | 平台 | 看什么 | 入口 |
|--------|------|--------|------|
| P0 | **小红书** | "空瓶记"、"成分党"测评、 before/after对比 | 搜索"XX成分""XX产品 空瓶" |
| P0 | **美丽修行** | 成分查询热度、产品评分变化 | APP内搜索 |
| P1 | **什么值得买** | 护肤品好价爆料、开箱测评 | 搜索"XX护肤品" |
| P1 | **天猫/京东榜单** | 品类销量排行、新品榜 | APP内榜单页面 |
| P2 | **知乎** | 成分科普、长期使用反馈 | 搜索"XX成分有用吗" |
| P2 | **品牌直播间** | 主播话术、主推品变化 | 抖音/淘宝直播间 |

### 每日5分钟流程

```
Day Check：

1. 打开小红书，搜索"护肤品推荐"，筛选"最热"
   └─ 记录：前10篇笔记主推什么成分/品牌？
   └─ 记录："早C晚A""以油养肤"等新概念是否出现？

2. 打开美丽修行，看"热门搜索"TOP 10
   └─ 记录：用户在主动查什么成分？

3. 打开什么值得买，看"美妆护肤"热榜
   └─ 记录：什么在打折？折扣力度多大？
```

### 每周30分钟深度扫描

```
Week Scan：

1. 成分热度追踪（10分钟）
   ├─ 小红书搜索热门成分（如烟酰胺、A醇、胜肽、麦角硫因）
   ├─ 对比各成分的笔记数增速
   ├─ 美丽修行查成分安全性和功效论文数量
   └─ 判断：哪个成分正在从"小众"走向"大众"？

2. 品牌动作分析（10分钟）
   ├─ 看雅诗兰黛、欧莱雅、宝洁等大集团的新品发布
   ├─ 看国货品牌（珀莱雅、薇诺娜、华熙生物）的新品动态
   ├─ 追踪"成分党"品牌的崛起（如The Ordinary、Paula's Choice）
   └─ 判断：品牌是否在押注某个新成分？

3. 价格周期观察（5分钟）
   ├─ 对比中免日上 vs 专柜 vs 代购的价格
   ├─ 追踪大促节点的折扣力度变化（双11是否比往年力度小？）
   └─ 记录：套装是否是最佳入手时机？

4. 更新追踪表（5分钟）
```

### 关键指标与阈值

| 指标 | 低热度 | 中热度 | 高热度 | 过热/泡沫 |
|------|--------|--------|--------|----------|
| 小红书周新增笔记 | <100 | 100-500 | 500-2000 | >2000 |
| 美丽修行搜索排名 | >50 | 20-50 | 10-20 | TOP 10 |
| 天猫品类排名变化 | 无变化 | 上升3位内 | 冲入前10 | 蝉联前3 |
| 成分科普视频量 | <10/周 | 10-50/周 | 50-200/周 | >200/周 |
| 负面反馈比例 | <5% | 5-10% | 10-20% | >20%（翻车信号） |

### 男女交叉追踪点

- **男性护肤增速**：对比"男士护肤"和"女士护肤"的搜索增速，判断男性赛道是否爆发
- **共用产品**：追踪SK-II神仙水、La Mer面霜等"男女通用"产品的男性用户增长
- **送礼套装**：追踪情人节/父亲节/母亲节套装的内容热度差异

## 品类四：手表（Watches）

### 为什么手表值得追踪

手表是**信息最不对称、圈层最封闭、保值率差异最大**的品类。热度信号往往先在封闭论坛出现，数月后才扩散到大众。

### 信号源清单

| 优先级 | 平台 | 看什么 | 入口 |
|--------|------|--------|------|
| P0 | **腕表之家论坛** | 表友讨论、晒表、行情交流 | xbiao.com |
| P0 | **Chrono24** | 全球行情价格、交易量 | chrono24.com |
| P1 | **Hodinkee** | 行业新闻、新品评测、趋势分析 | hodinkee.com |
| P1 | **小红书** | 入门表推荐、实际上手效果 | 搜索"男表推荐""XX表测评" |
| P2 | **品牌官网** | 新品发布、涨价通知、停产公告 | 各品牌官网新闻页 |
| P2 | **得物** | 时尚表款价格（如卡西欧/G-Shock限量） | APP内搜索 |

### 每日5分钟流程

```
Day Check：

1. 打开腕表之家，看论坛首页热帖
   └─ 记录：表友在讨论什么品牌/型号？
   └─ 记录：是否有"涨价""停产""新配色"等关键词？

2. 打开Chrono24，看首页推荐和价格变动
   └─ 记录：热门款价格是否在波动？

3. 打开小红书，搜索"手表推荐"
   └─ 记录：入门级用户在关注什么？（与表友圈可能有差异）
```

### 每周30分钟深度扫描

```
Week Scan：

1. 行情追踪（10分钟）
   ├─ 查3-5款追踪表中的热门表在Chrono24的价格
   ├─ 对比上周价格，计算涨跌幅
   ├─ 看腕表之家"行情交流"区的最新讨论
   └─ 判断：价格变动是因为供需变化，还是市场情绪？

2. 新品与政策（10分钟）
   ├─ 看Hodinkee本周文章，了解全球表圈动态
   ├─ 看品牌官网是否有涨价/停产/新品发布
   ├─ 追踪劳力士、百达翡丽、AP的配货政策变化
   └─ 判断：政策变化对二级市场的影响

3. 圈层情绪（5分钟）
   ├─ 看表友对某品牌/某款的情绪倾向
   ├─ 区分：真实玩家声音 vs 投机炒家声音
   └─ 判断：当前市场情绪是贪婪、恐惧还是理性？

4. 更新追踪表（5分钟）
```

### 关键指标与阈值

| 指标 | 低热度 | 中热度 | 高热度 | 过热/泡沫 |
|------|--------|--------|--------|----------|
| 腕表之家周讨论帖数 | <10 | 10-50 | 50-200 | >200 |
| Chrono24价格周波动 | <2% | 2-5% | 5-10% | >10% |
| 二级市场溢价率 | 折价 | 原价±5% | 溢价5-20% | 溢价>50% |
| 专柜等待时间 | 随时可买 | 1-3个月 | 3-12个月 | >1年或停止登记 |
| 小红书入门推荐提及率 | <5% | 5-15% | 15-30% | >30%（大众认知渗透） |

### 男女交叉追踪点

- **女性表友增长**：追踪"女表推荐""女生戴男表"话题，判断女性是否进入表圈
- **情侣对表**：追踪品牌推出的对表系列的市场反馈
- **表径偏好变化**：女性是否偏好大表径（>36mm），男性是否接受小表径（<38mm）

## 品类五：球鞋/潮流鞋（Sneakers）

### 为什么球鞋值得追踪

球鞋是**热度周期最短、价格波动最剧烈、圈层文化最浓**的品类。今天的"神鞋"，三个月后可能"破发"。

### 信号源清单

| 优先级 | 平台 | 看什么 | 入口 |
|--------|------|--------|------|
| P0 | **得物** | 价格曲线、交易量、涨幅榜 | APP首页"涨跌榜" |
| P0 | **SNKRS** | Nike限量发售日历 | APP内发售页面 |
| P1 | **小红书** | 实际上脚图、搭配方案 | 搜索"XX鞋穿搭" |
| P1 | **StockX/GOAT** | 国际市场行情 | stockx.com / goat.com |
| P2 | **Hypebeast/Highsnobiety** | 潮流新闻、联名信息 | 官网/公众号 |
| P2 | **B站** | 球鞋测评、开箱 | 搜索"XX鞋测评" |

### 每日5分钟流程

```
Day Check：

1. 打开得物，看"今日涨跌榜"TOP 20
   └─ 记录：哪几款在暴涨？哪几款在暴跌？
   └─ 记录：暴涨/暴跌的原因是什么？（发售、明星上脚、补货？）

2. 打开SNKRS/Confirmed，看近期发售日历
   └─ 记录：未来7天有什么限量款发售？
   └─ 记录：已发售款的二级市场表现

3. 打开小红书，搜索"球鞋推荐"
   └─ 记录：大众用户在关注什么？（可能与炒鞋圈完全不同）
```

### 每周30分钟深度扫描

```
Week Scan：

1. 市场扫描（10分钟）
   ├─ 看得物/StockX本周涨幅最大的10款鞋
   ├─ 看跌幅最大的10款鞋
   ├─ 分析：涨跌与什么事件相关？（明星上脚？补货？季节变化？）
   └─ 判断：当前市场处于炒作期还是冷却期？

2. 发售复盘（10分钟）
   ├─ 回顾本周发售的限量款
   ├─ 查每款的：发售价 → 首日市价 → 当前市价
   ├─ 计算：破发率（破发款数/总发售款数）
   └─ 判断：市场是否过热？（破发率>50%=冷却信号）

3. 趋势前瞻（5分钟）
   ├─ 看Hypebeast下周发售预告
   ├─ 追踪明星/网红上脚动态
   ├─ 关注品牌财报中提到的"重点品类"
   └─ 判断：什么品类/配色可能在下一波爆发？

4. 更新追踪表（5分钟）
```

### 关键指标与阈值

| 指标 | 低热度 | 中热度 | 高热度 | 过热/泡沫 |
|------|--------|--------|--------|----------|
| 得物日交易量 | <100双 | 100-500双 | 500-2000双 | >2000双 |
| 周价格波动 | <5% | 5-15% | 15-30% | >50% |
| 小红书穿搭笔记周增 | <50 | 50-200 | 200-1000 | >1000 |

## 六、跨品类综合观察：发现结构性机会

当你同时追踪多个品类时，你会发现**跨品类的联动信号**：

| 跨品类现象 | 信号组合 | 意味着什么 |
|-----------|---------|-----------|
| **香水+包袋同品牌联动** | 某品牌香水和包同时在热 | 品牌整体势能上升，不只是单品运气 |
| **护肤品+彩妆同集团联动** | 某集团多品牌同时推新 | 集团层面有大动作（新成分、新营销预算） |
| **手表+球鞋同时降温** | 两个"男性玩具"同时冷 | 男性可支配收入可能承压，或潮流周期切换 |
| **中性标签跨品类扩散** | 香水、包、鞋同时出现"无性别"趋势 | 社会审美正在发生结构性变化 |
| **送礼内容跨品类集中** | 多个品类同时出现"送XX"内容爆发 | 临近节日（情人节、520、七夕、圣诞） |

## 七、你的第一周执行计划

### Day 1：选定1-2个主攻品类

### Month 1：验证你的判断

用你第一个月的观察，做一次预测（如"某款会火""某款会降温"），等一个月验证。

## 八、总结：从"看数据"到"有直觉"

> 追踪热度的终极目标，不是成为一个数据库，而是培养一种**"市场直觉"**——当你走进一个商场、打开一个APP、参加一个聚会时，你能自动感知到"什么正在变化"。
>
> 这种直觉无法从书本上学到，只能通过持续的、有意识的观察来培养。
>
> 每天10分钟，坚持3个月，你会惊讶于自己的变化。
